2) University treasure hunt is a game to test the memory and the ability to find hidden quests.
Game is primarily designed for the faculty and the students of university. They would have to 
find the answers which are based on the actual facts in their surounding.

3) Project is developed with Android studio v2.3.2. Project can only be compiled with android studio
having correct gradle file dependencies.

4) Important files are present in these directories:

* Java Code: University Treasure Hunt\app\src\main\java\com\usman\treasurehuntgame
* APK: University Treasure Hunt\app\build\outputs\apk
